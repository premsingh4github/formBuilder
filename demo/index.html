jQuery formBuilder

A jQuery plugin for drag and drop form creation

Download this project as a .zip file Download this project as a tar.gz file

Usage

To start building forms with this plugin simply call formBuilder() on the textarea you would like to make your editor. FormBuilder takes a number of options and is translatable through these options.

jQuery(document).ready(function($) {
  'use strict';
  $(document.getElementById('form-builder-template')).formBuilder();
});

Render The Form

Add some fields to the formBuilder above and render them here.

Once you've created a form you'll need a way to render it. formRender is the companion plugin to formBuilder that renders the form data. To use you''ll call formRender on the textarea containing your saved data like so:

var template = document.getElementById('form-builder-template');
$(template).formRender();

By default formRender() will replace the textarea with the rendered form but passing a container option to the plugin will render the form to the specified jQuery object. Example:

var template = document.getElementById('form-builder-template'),
formContainer = document.getElementById('rendered-form'),
formRenderOpts = { container: $(formContainer) };
$(template).formRender(formRenderOpts);
